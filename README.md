## 🛠️ **Stack Tecnológica**

### **Frontend & UI/UX**
- **Next.js 14.2.6** - Framework React com SSR e otimizações avançadas
- **React 18** - Biblioteca para interfaces interativas e responsivas
- **TypeScript 5** - Tipagem estática para código mais robusto e escalável
- **Tailwind CSS 3.4** - Framework CSS utility-first com animações customizadas
- **Framer Motion 11.18** - Biblioteca de animações para experiências fluidas

### **Animações & Efeitos Visuais**
- **Animações CSS personalizadas** com keyframes avançados
- **Transições suaves** entre componentes e páginas
- **Efeitos de parallax** otimizados para performance
- **Grid responsivo** com breakpoints inteligentes

### **Performance & Otimização**
- **Lazy loading** de componentes e imagens
- **Code splitting** automático com Next.js
- **Otimizações de bundle** e tree shaking
- **Animações baseadas em viewport** para melhor performance

## 🚀 **Como Executar o Projeto**

### **Pré-requisitos**
- Node.js 18+ 
- npm ou yarn

### **Instalação**
```bash
# Clone o repositório
git clone https://github.com/seu-usuario/portfolio-modern.git

# Entre na pasta
cd portfolio-modern

# Instale as dependências
npm install

# Execute em modo desenvolvimento
npm run dev
```

### **Scripts Disponíveis**
```bash
npm run dev      # Servidor de desenvolvimento
npm run build    # Build de produção
npm run start    # Servidor de produção
npm run lint     # Verificação de código
```
